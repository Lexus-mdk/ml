При запуске сервера в консоли появится ссылка(http://127.0.0.1:5000/), по ней вы перейдете на начальную страницу.
Далее для работы с API нужно прописать команду после порта локального сервера.
Например:
http://127.0.0.1:5000/model/prediction/example.csv/ - перейдя по данной ссылке API сделает предсказание по файлу "example.csv" и выведет на экран результат

Команды:
/model/prediction/(аргументы)/ - запрос на предсказание модели
    * аргументом является название файла с расширением .csv
    * файл по которому нужно предсказывать должен находиться в папке data/, которая находится в корневой директории
    * пример файла находится в папке data/ с названием example.csv
